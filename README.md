# EasyPixel
EasyPixel es una aplicación que permite subir imágenes desde cualquier dispositivo y pixelarlas mediante una barra deslizadora, estas imágenes son publicadas en un catálogo donde los demás usuarios pueden acceder a ellas y descargarlas. La aplicación está conectada a una base de datos en Azure donde se almacenan la información necesariapara las páginas de detalle.
